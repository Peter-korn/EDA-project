# Netflix Shows EDA Project

Репозиторий содержит разведочный анализ данных (Exploratory Data Analysis) базы данных сериалов и шоу Netflix. В ходе проекта была проведена предобработка данных, очистка от дубликатов, обработка аномалий и визуализация ключевых характеристик контента.

## 📋 Описание проекта
Цель проекта — изучить структуру библиотеки Netflix, проанализировать распределение возрастных рейтингов и выявить закономерности в данных с помощью Python.

### Основные этапы работы
1. **Загрузка данных:** Использование датасета `NetflixShows.xlsx`.
2. **Очистка данных:** Удаление дубликатов, анализ спецсимволов в названиях и валидация дат выпуска.
3. **Преобразование признаков:** Перекодировка возрастных рейтингов (разделение категорий G/TV-G и NR/UR) для точного анализа.
4. **Анализ и Визуализация:** Исследование распределения контента и построение графиков.

## 🛠 Технологический стек
Проект реализован на языке **Python** с использованием библиотек:
* `pandas` — манипуляции с данными.
* `numpy` — математические вычисления.
* `matplotlib` & `seaborn` — визуализация данных.

## 🔍 Ключевые инсайты и результаты

### Основные выводы анализа:
* **Превосходство сериалов:** Средний пользовательский рейтинг сериалов стабильно выше, чем у полнометражных фильмов.
* **Смена целевой аудитории:** В последние годы стриминг делает явный акцент на расширение библиотеки контента для взрослой и подростковой аудитории.
* **Географические лидеры:** Самые высокие рейтинги среди сериалов демонстрирует контент из Колумбии, Ирландии, Мексики, Великобритании и Австралии.
* **Жанровые предпочтения:** В топ-4 жанра с наивысшими оценками вошли триллеры, криминальные драмы, мистика и романтика.
* **Тренд оценок:** Наблюдается эффект «инфляции оценок» — средние баллы пользователей имеют тенденцию к росту со временем.

### Технические результаты:
* **Обработка аномалий:** Выявлены и сохранены тайтлы со спецсимволами (например, *Pokémon: Indigo League*), которые были определены как валидные данные.
* **Уточнение рейтингов:** Скорректированы числовые коды `ratingDescription` (например, для TV-G и UR), что позволило устранить пересечения категорий.
* **Валидация дат:** Подтверждена достоверность временного диапазона данных (релизы с 1888 по 2017 гг.).
