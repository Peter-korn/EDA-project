# Netflix Shows EDA Project
Репозиторий содержит разведочный анализ данных (Exploratory Data Analysis) базы данных сериалов и шоу Netflix. В ходе проекта была проведена предобработка данных, очистка от дубликатов, обработка аномалий и визуализация ключевых характеристик контента.

# Описание проекта
Цель проекта — изучить структуру библиотеки Netflix, проанализировать распределение возрастных рейтингов и выявить закономерности в данных с помощью Python.

## Основные этапы работы:

Загрузка данных: Использование датасета NetflixShows.xlsx.

Очистка данных:

Удаление дубликатов строк.

Анализ некорректных символов в названиях (тайтлах) и проверка их на валидность.

Валидация годов выпуска (проверка на отсутствие "будущих" или неправдоподобно старых дат).

Преобразование признаков: Перекодировка описаний возрастных рейтингов (например, разделение категорий G/TV-G и NR/UR) для устранения пересечений.

Анализ дубликатов: Исследование частоты повторения различных категорий контента.

Визуализация: Построение графиков (Scatter plots и др.) для оценки распределения рейтингов.

🛠 Технологический стек
Проект реализован на языке Python с использованием следующих библиотек:

pandas — для манипуляций с данными и анализа таблиц.

numpy — для работы с массивами.

matplotlib & seaborn — для создания визуализаций и графиков.

📊 Ключевые результаты
Обработка аномалий: В ходе анализа были выявлены тайтлы со спецсимволами (например, Pokémon: Indigo League), которые после проверки были признаны корректными данными, а не выбросами.

Уточнение рейтингов: Была проведена корректировка числовых кодов ratingDescription. Например, рейтинги TV-G и UR получили уникальные идентификаторы (38 и 130 соответственно), чтобы не пересекаться с похожими категориями.

Release Year: Проверка показала, что все данные находятся в корректном временном диапазоне (нет выпусков позже 2017 года или ранее 1888 года).
